# Study Buddy

**Study Buddy** is a document-based learning web application that converts PDFs and text files into interactive study materials like summaries, quizzes, flashcards, and a quiz game.

This project is built as a **demo-ready application** with smooth UI flow and guided interactions.

---

## Features

* Upload PDF or TXT documents
* Generate structured summaries
* Create multiple-choice quizzes
* View flashcards
* Play quiz-based learning game
* Ask questions from the document

---

## Tech Stack

* Frontend: React (Vite)
* Backend: FastAPI
* Styling: Tailwind CSS
